## توضیح کلی  پروژه 
 این پروژه شامل طراحی یک وب‌سایت خدمات خانه‌داری است که مشتریان بتوانند به‌صورت آنلاین درخواست خدمات نظافت و امور منزل را ثبت کرده و ارائه‌دهندگان خدمات به آن پاسخ دهند. هدف، ایجاد یک نمونه اولیه ساده و کاربرپسند برای دیجیتالی‌کردن خدمات سنتی خانه‌داری است.وایجاد دسترسی آسان و سریع به خدمات منزل



## اهداف پروژه

_کاهش زمان و هزینه مراجعه حضوری
_ایجاد بستری برای توسعه کسب‌وکارهای کوچک
